# An example of density plot (df3 format)

## Summary

Sample files for povray using density plot (df3 format).

#Usage

    $ make clean
    $ make

## Files

* [df3.rb](df3.rb)
  Ruby script which generates df3 file (test.df3).

* [df3.pov](df3.pov)
  Povray file for density plot.

* [df3.png](df3.png)
  An output image.
